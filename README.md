# Online-Marketplace

My e-commerce project is a web platform designed to help sellers easily upload and showcase their products, while buyers can browse, view, and purchase items online.

## Features

- User Authentication
- Item Posting
- Item Browsing
- Wishlist
- Payment
- Inbox seller
